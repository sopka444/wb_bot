## 🎯 Описание

Мощный Telegram бот для селлеров Wildberries, который предоставляет:
- 📊 Аналитику продаж и остатков в реальном времени
- 📈 Визуализацию данных в виде графиков
- 🔍 Поиск информации по конкретным товарам
- 📸 AI-анализ фотографий товаров с рекомендациями
- 📅 Гибкие отчеты за разные периоды

## ✨ Возможности

### 📊 **Аналитика продаж**
- Общее количество продаж
- Выручка и средний чек
- Динамика по дням
- Топ товаров по выручке

### 📦 **Управление остатками**
- Текущие остатки на складе
- Товары с критически низким запасом
- Статистика по уникальным товарам

### 📈 **Визуализация**
- Интерактивные графики продаж
- Динамика выручки
- Тренды и паттерны

### 🤖 **AI-анализ фото (Gemini)**
- Оценка качества изображения
- Анализ инфографики
- Рекомендации по улучшению
- Соответствие категории товара

### 🔍 **Поиск и отчеты**
- Поиск по артикулу товара
- Отчеты за произвольные периоды
- Детальная информация по товарам

⚙️ Настройка
1. Получение Telegram Bot Token
Напишите @BotFather в Telegram

Отправьте команду /newbot

Получите токен вида 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

2. Получение Wildberries API Token
Войдите в личный кабинет Wildberries

Перейдите в раздел "Настройки" → "Доступ к API"

Создайте новый токен с правами на статистику

Скопируйте токен

3. Получение Google Gemini API Key
Перейдите на makersuite.google.com/app/apikey

🚀 Запуск

Бот должен ответить на команду /start в Telegram
