## 🎯 Описание проекта

Данный проект решает задачу бинарной классификации комментариев на два класса:
- **0** - Нетоксичный комментарий
- **1** - Токсичный комментарий (оскорбления, грубость, агрессия)

Модель основана на архитектуре с двунаправленными GRU слоями и механизмом объединения усредненного и максимального пулинга для лучшего извлечения признаков.
