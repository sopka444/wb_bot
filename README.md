# WB Thunder Bot ⚡️
WB Thunder Bot — это мощный и безопасный Telegram-бот для профессиональной аналитики Wildberries. Он объединяет данные с неограниченного количества магазинов в едином интерфейсе, позволяя маркетплейс-предпринимателям и селлерам принимать взвешенные управленческие решения на основе точных данных.
Основные возможности 🌩️
# 👥 Мультиаккаунтность
Подключение неограниченного количества магазинов Wildberries.

Мгновенное переключение между магазинами внутри бота.

Сводная аналитика: Просмотр общих показателей по всем подключенным аккаунтам одновременно.

Единый интерфейс для управления всеми проектами.
